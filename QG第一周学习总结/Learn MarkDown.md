## 四、解决方法
1. **keil5解决办法**
   - 通过重启单片机进行硬件与软件的握手
   - 在网上查找资料后，直接将51文件夹放置keil5文件夹内
   - 尝试将芯片换方向接入后不发热，故推断是芯片短路导致发热
2. **GitHub及git解决方法**
   - 通过学习B站视频 [给傻子的git教程] 学会了基本的上传代码
   - 通过开启vpn或开启局部代理解决GitHub无法打开的问题
   - - 通过输入：
     - git branch --set-upstream-to=origin/develop develop
     - git pull
     - 来解决无响应分支的问题
   - - 通过输入：
     - git pull ==origin master== –allow-unrelated-histories
     - git push ==origin master==:==master==
     - 来解决合并两个独立启动仓库的历史
3. **MarkDown学习**
   - 通过学习B站[【全网最详细Markdown入门使用教程】40分钟教会你用免费的vscode做出好看的笔记]来学习
   - 通过增加修改名字如[first save]来成功提交
